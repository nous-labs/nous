# Getting Started with Qubic TypeScript SDK

The Qubic TypeScript SDK (QTS) is a comprehensive library for integrating with the Qubic blockchain. It provides type-safe, modern APIs for querying blockchain data, interacting with smart contracts, and managing transactions.

<Callout type="info">
  **Universal Support**: Works in browsers, Node.js, Bun, and Deno with full TypeScript support.
</Callout>

### Create a Client

The simplest way to get started is using the unified client that provides access to all services:

```typescript
import { createQubicClient } from '@nvlabs/qts';

const qubic = createQubicClient();
```

### Get Current Tick

Retrieve the current network tick information:

```typescript
const { tickInfo } = await qubic.live.getTickInfo();
console.log(`Current tick: ${tickInfo.tick}`);
console.log(`Epoch: ${tickInfo.epoch}`);
```

### Check Account Balance

Query the balance of any Qubic identity:

```typescript
const identity = 'BZVMIJXDWZQJWTFVEBPCJVFZDHXICRCLUVDUPKQGIJAFLEZCMMLUQHTXEXWA';
const { balance } = await qubic.live.getBalance(identity);
console.log(`Balance: ${balance.balance} QUBIC`);
```

### Get Transaction History

Retrieve transaction history with pagination and filters:

```typescript
const { transactions } = await qubic.query.getTransactionsForIdentity(
  identity,
  {
    pagination: { offset: 0, size: 10 }
  }
);

transactions.forEach(tx => {
  console.log(`${tx.source} ‚Üí ${tx.destination}: ${tx.amount}`);
});
```

## Key Features

### Type Safety

Full TypeScript definitions with strict mode support:

```typescript
import type { Transaction, Balance, TickInfo } from '@nvlabs/qts';

const processTransaction = (tx: Transaction) => {
  console.log(`From: ${tx.source}`);
  console.log(`To: ${tx.destination}`);
  console.log(`Amount: ${tx.amount}`);
};
```

### Tree-Shakeable

Import only what you need to keep your bundle size small:

```typescript
// Import specific clients
import { QubicLiveClient, QueryClient } from '@nvlabs/qts';

// Import utilities
import { encodeInt64LE, hexToBase64 } from '@nvlabs/qts';

// Import contract helpers
import { createQuery, parseResponse } from '@nvlabs/qts';
```

### Universal Compatibility

Works seamlessly across all JavaScript environments:

- **Browsers**: Chrome, Firefox, Safari, Edge
- **Node.js**: Version 18+
- **Bun**: Version 1.0+
- **Deno**: Latest version

### Smart Contract Support

Simplified smart contract querying with automatic serialization:

```typescript
import { createQuery, parseResponse, QUBIC_CONTRACTS } from '@nvlabs/qts';

const query = createQuery(QUBIC_CONTRACTS.QX, 1)
  .addInt32(100)
  .addInt64(1000n);

const response = await query.execute(qubic.live);
const parser = parseResponse(response.responseData);
const result = parser.readInt64();
```

## Core Concepts

### Three API Clients

QTS provides three specialized clients for different use cases:

1. **QubicLiveClient** (`qubic.live`)
   - Real-time network data
   - Current tick information
   - Balance queries
   - Smart contract interactions
   - Transaction broadcasting

2. **QueryClient** (`qubic.query`)
   - Historical transaction data
   - Advanced filtering and pagination
   - Tick data archives
   - Computor information

3. **ArchiveClient** (`qubic.archive`)
   - Legacy archive API
   - Historical data access
   - *Note: Some methods deprecated in favor of QueryClient*

### Error Handling

All API calls can throw `QubicApiError` with detailed information:

```typescript
try {
  const balance = await qubic.live.getBalance(identity);
  console.log('Balance:', balance.balance.balance);
} catch (error) {
  if (error.status === 404) {
    console.log('Identity not found');
  } else if (error.status === 408) {
    console.log('Request timeout - try again');
  } else {
    console.error('Error:', error.message);
  }
}
```

### Configuration

Customize client behavior with configuration options:

```typescript
const qubic = createQubicClient({
  // Custom timeouts
  timeout: 60000, // 60 seconds

  // Custom API endpoints
  liveUrl: 'https://custom-rpc.qubic.org',
  queryUrl: 'https://custom-api.qubic.org',

  // Custom headers
  headers: {
    'X-API-Key': 'your-api-key'
  }
});
```

## Common Patterns

### Monitor New Ticks

Watch for new ticks in real-time:

```typescript
let lastTick = 0;

setInterval(async () => {
  const { tickInfo } = await qubic.live.getTickInfo();
  if (tickInfo.tick > lastTick) {
    console.log(`New tick: ${tickInfo.tick}`);
    lastTick = tickInfo.tick;
  }
}, 1000);
```

### Paginate Through Transactions

Fetch all transactions with pagination:

```typescript
async function getAllTransactions(identity: string) {
  const transactions = [];
  let offset = 0;
  const pageSize = 100;

  while (true) {
    const result = await qubic.query.getTransactionsForIdentity(
      identity,
      { pagination: { offset, size: pageSize } }
    );

    transactions.push(...result.transactions);

    if (result.transactions.length < pageSize) break;
    offset += pageSize;
  }

  return transactions;
}
```

### Query Multiple Assets

Fetch all asset types for an identity:

```typescript
const identity = 'YOUR_IDENTITY_HERE';

const [issued, owned, possessed] = await Promise.all([
  qubic.live.getIssuedAssets(identity),
  qubic.live.getOwnedAssets(identity),
  qubic.live.getPossessedAssets(identity)
]);

console.log('Total assets:', {
  issued: issued.issuedAssets.length,
  owned: owned.ownedAssets.length,
  possessed: possessed.possessedAssets.length
});
```
