# CCF

Community Consensus Fund governance.

## Available hooks

```ts
useCCFLatestTransfers()
useCCFProposal()
useCCFProposalFee()
useCCFProposalIndices()
useCCFVote()
useCCFVotingResults()
```

Each hook lives in `@nouslabs/react/query` and wraps the underlying `@nouslabs/sdk` helpers.
