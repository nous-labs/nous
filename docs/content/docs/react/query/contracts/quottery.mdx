# Quottery

Betting pools and wagering results.

## Available hooks

```ts
useQuotteryActiveBet()
useQuotteryBasicInfo()
useQuotteryBetByCreator()
useQuotteryBetInfo()
useQuotteryBetOptionDetail()
useQuotteryQuery()
```

Each hook lives in `@nouslabs/react/query` and wraps the underlying `@nouslabs/sdk` helpers.
