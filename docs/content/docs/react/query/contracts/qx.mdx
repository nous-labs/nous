# QX Exchange

Orderbook, fees, and marketplace insights.

## Available hooks

```ts
useQXAssetAskOrders()
useQXAssetBidOrders()
useQXEntityAskOrders()
useQXEntityBidOrders()
useQXFees()
useQXQuery()
```

Each hook lives in `@nouslabs/react/query` and wraps the underlying `@nouslabs/sdk` helpers.
