# Qswap

Automated market maker pools and quoting.

## Available hooks

```ts
useQswapFees()
useQswapLiquidityOf()
useQswapPoolBasicState()
useQswapQuoteExactAssetInput()
useQswapQuoteExactAssetOutput()
useQswapQuoteExactQuInput()
useQswapQuoteExactQuOutput()
useQswapTeamInfo()
```

Each hook lives in `@nouslabs/react/query` and wraps the underlying `@nouslabs/sdk` helpers.
