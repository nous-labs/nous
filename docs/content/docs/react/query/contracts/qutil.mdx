# Qutil

Governance polls, fee schedules, and asset share insights.

## Available hooks

```ts
useQutilCurrentPollId()
useQutilCurrentResult()
useQutilPollInfo()
useQutilPollsByCreator()
useQutilSendToManyV1Fee()
useQutilTotalAssetShares()
```

Each hook lives in `@nouslabs/react/query` and wraps the underlying `@nouslabs/sdk` helpers.
